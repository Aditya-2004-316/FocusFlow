# FocusFlow Timer Extension - Quick Start

## 30-Second Setup

### Chrome / Edge / Brave

1. **Open extensions page**

    - Chrome: `chrome://extensions`
    - Edge: `edge://extensions`
    - Brave: `brave://extensions`

2. **Toggle Developer mode** (top-right corner)

3. **Click "Load unpacked"**

4. **Select the `extension` folder** from `d:\FocusFlow\extension`

5. **Done!** The timer appears on all tabs instantly

### Firefox

1. **Open debugging page**: `about:debugging`

2. **Click "This Firefox"** (left sidebar)

3. **Click "Load Temporary Add-on"**

4. **Select `manifest.json`** from `d:\FocusFlow\extension`

5. **Done!** The timer appears on all tabs instantly

## Permissions

The extension asks for only **2 permissions**:

-   `storage` - To save your timer state
-   `scripting` - To inject the timer UI

**Privacy guarantee**: No data collection, no tracking, no page content reading.
